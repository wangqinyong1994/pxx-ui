## API

### props

| 属性             | 说明             | 类型               | 默认值              |
| ---------------- | ---------------- | ------------------ | ------------------- |
| title            | 标题             | `string            | React.ReactElement` | - |
| wrapClassName    | modal 类名       | string             | -                   |
| visible          | 显示隐藏         | boolean            | -                   |
| renderContainer  | 内容渲染         | React.ReactElement | -                   |
| onConfirmHandler | 确认回调函数     | Function           | -                   |
| onTriggerHander  | 切换显示隐藏函数 | Function           | -                   |
